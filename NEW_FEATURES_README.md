# 新功能使用指南

## 🆕 新增的4个功能

### 1. ⏰ 时光计时器 (`/countdown`)
**功能说明：**
- 倒计时：计算距离未来某个日期还有多久（如：下次见面、生日、纪念日）
- 正计时：计算从某个日期开始已经过了多久（如：在一起多少天）
- 实时更新：精确到秒的动态显示

**使用方法：**
1. 点击"添加计时"按钮
2. 填写标题（如"下次见面"、"在一起"、"生日"）
3. 选择日期和时间
4. 选择类型：倒计时（未来事件）或正计时（已过天数）
5. 选择喜欢的图标
6. 点击"添加计时"保存

**特色：**
- 支持自定义图标（⏰、💝、🎂、🎉等）
- 实时动态显示天/时/分/秒
- 可以创建多个计时器
- 数据永久保存在Supabase

---

### 2. 📅 共享日程 (`/schedule`)
**功能说明：**
- 记录两人的约会计划
- 设置提醒时间
- 标记日程状态（即将到来/已完成/已取消）

**使用方法：**
1. 点击"添加日程"按钮
2. 填写日程标题和详细描述
3. 选择日期、时间和地点
4. 设置提前提醒时间（15分钟/30分钟/1小时/2小时/1天）
5. 填写创建者名字
6. 保存后可以标记为"完成"或"取消"

**筛选功能：**
- 全部：查看所有日程
- 即将到来：只看未完成的日程
- 已完成：查看完成的日程

**特色：**
- 🔔 即将到来的日程会显示铃铛提醒
- 📍 支持添加地点信息
- ⏰ 可自定义提醒时间
- 多状态管理

---

### 3. 🎁 时光胶囊 (`/time-capsule`)
**功能说明：**
- 写给未来的信
- 设定开启日期，在此之前内容会被"封存"
- 到达指定日期后可以开启查看

**使用方法：**
1. 点击"创建胶囊"按钮
2. 给胶囊取个名字
3. 写下想对未来说的话
4. 填写创建者和收件人（可选）
5. 设定开启日期（必须是未来的时间）
6. 封存后，在开启日期前内容会被锁定

**开启胶囊：**
- 未到开启时间：显示🔒和倒计时
- 到达开启时间：显示🔓，可以点击"开启胶囊"查看内容
- 开启后：显示📖和完整内容

**特色：**
- 到期可开启的胶囊会有动画提示
- 支持写给特定收件人
- 记录创建和开启时间
- 充满仪式感的设计

---

### 4. 📖 恋爱日记 (`/diary`)
**功能说明：**
- 共同编辑的日记本
- 记录每天发生的事情和心情
- 支持筛选和编辑

**使用方法：**
1. 点击"写日记"按钮
2. 选择日期（默认今天）
3. 填写标题（如"第一次约会"、"平凡的一天"）
4. 选择今天的心情（😊😍🥰😘🤗😌🥳😭😤🤔）
5. 写下日记内容
6. 填写作者名字
7. 保存后可以随时编辑或删除

**功能亮点：**
- 📅 按日期筛选：快速查找某一天的日记
- ✏️ 可编辑：点击铅笔图标编辑已有日记
- 😊 心情记录：10种心情emoji可选
- ✍️ 作者标记：知道是谁写的
- 🕐 自动记录更新时间

## 📊 数据库配置

### 重要：在使用前必须在Supabase执行SQL！

1. 登录你的 [Supabase Dashboard](https://supabase.com/dashboard)
2. 选择你的项目
3. 进入 SQL Editor
4. 打开 `supabase-new-tables.sql` 文件
5. 复制全部内容粘贴到SQL Editor
6. 点击"Run"执行

**创建的表：**
- `countdowns` - 时光计时器数据
- `schedules` - 共享日程数据
- `time_capsules` - 时光胶囊数据
- `diary_entries` - 恋爱日记数据

**安全策略：**
所有表都启用了行级安全策略(RLS)，并设置为公开读写访问。
